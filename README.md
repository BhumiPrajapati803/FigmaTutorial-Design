# Tutorial Design Project

This repository contains the **Tutorial design project** files exported from Figma. It includes both the Figma source file and a PDF preview for easy viewing.

---

## Files

- [Tutorial.pdf](./Tutorial.pdf) – PDF version of the design for quick preview.
- [Tutorial.fig](./Tutorial.fig) – Figma source file for editing in Figma.

---

## Preview

No images were provided for previews. You can open the PDF to see the design.

---

## How to View

1. **PDF Preview:** Click on the `Tutorial.pdf` link to view the design directly in your browser.  
2. **Figma Design:** Open `Tutorial.fig` in Figma to view and edit the original design.  
   - You can upload the `.fig` file to [Figma](https://www.figma.com/) to view and collaborate.

---

## Description

This project contains the full UI/UX design of the Tutorial project. You can use the PDF for reference or open the Figma file to continue designing, prototyping, or collaborating with your team.

---

## Notes

- Make sure you have a Figma account to open the `.fig` file.  
- PDF is viewable without any special software.
